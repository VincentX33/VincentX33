Now its time to learn about absolute positioning. Absolute positions the element relative to the nearest positioned ancestor ie, it depends on its closest parent. This makes it rather dangerous, as if one changes, others change too. If I give it absolute and make top and left 0, it flies up to its nearest ancestors top ie right on top. But if I give it top = 100em, it will come down.

Now its time to understand the sticky position property. Often used for popups and email subscription boxes, they retain a particular position in the container until a certain scroll position is reached at which point they behave like fixed position elements.

The div contained sticks to the scrollbar and refuses to leave, scrolling along nicely. Of course, for it to work properly, we need both position sticky and then the top property telling where the sticky element shud float. I used top 1em.
Sticky elements don't overflow their container elements ie if you scroll till the container is gone past, the sticky element stays where it is.
